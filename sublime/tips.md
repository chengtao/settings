sublime text tips

## mac settings
```
defaults write com.sublimetext.2 ApplePressAndHoldEnabled -bool false
defaults write -g ApplePressAndHoldEnabled -bool false
```

* command-d command-k
* regex: command-i, command-alt-r, ```(?<=<h2>).+(?=</h2>)```, alt-enter
